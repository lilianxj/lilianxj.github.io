再见，再见！开智学堂信息分析课程三期结业复盘

Posted on 2019-01-27 | In 复盘笔记 | | Visitors:

背景：是结束也是一个新的开始

2018年的最后一个月，我走到人生的一个最低谷——迷失在人间，一条无人能够替我走的道路上。

回想发现，我是一个极其懒惰的人，从小是一个听话、乖巧且循规蹈矩的孩子。我永远在别人看得见的地方按照规则走，只有在一个人独处时，才能抛开所有别人的要求、眼光、规矩和习惯，自自在在地做自己，以至于产生两个严重的结果：一、我越来越孤僻，只喜欢一个人呆着，也害怕处于一个有我以外的人的空间；二、结果长期耍自闭、只在自己习惯的领域玩耍，导致没有真正养成一些身为人该有的好习惯（相反的，累积了数之不尽的坏习惯，如超爱看电视，连续剧、动漫小说等，不能自拔，20年的时光都耗在上面了……）

也许，我这么做，只是因为害怕麻烦，想要避开任何引人不快、耗费精力的事物而已吧？

在学习上，我也贯彻了一如既往的作风——从来不费心思去学习困难、麻烦的事，不懂的地方未曾想过要去动手查询，补充知识，只一味地把要考试的内容背起来就去考，想复印机一样，还为了结果不错沾沾自喜，抱持僵化型思维、完美主义的心态，成为了愚蠢的学习者而不自知。这导致我在任何学习上都停留在入门的平台期，怎么学习和练习都上不去。我也没有继续深入，而是到处寻找新的刺激。

我想，这也是一种懒惰的表现吧？

糟心的是，这种做法让我人生的前二十几年顺风顺水，看着也挺不错的，生活和学习上没有经历过太多的低谷及考验。

最大的矛盾，在我18~25岁期间才缓缓爆发。不用掐指一算，都明显看见这几个大矛盾是发生在校园生活和走出社会的交接口。

2018年下半年，是少年在大学毕业后一段充斥希望和绝望的漂流，然而，这段旅程一点也不奇幻。徘徊在“坚持自己的梦想，但却无能为力、被现实的困境啃噬”及“放弃自己的梦想，被现实淹没，随波逐流、庸庸碌碌度过一生”两个选项，并且，在努力输出作品、与自己个性和习惯上的缺失作战、与现实生活的压力作战等三方面摆荡，心力交瘁。

如果不写作，我想做什么呢？我发现自己最感兴趣的，是线上教育。

远远地望着自己追了两年的开智，那么远，那么近，看起来十分美好。如流星瞬间划过黑夜的天际，心中顷刻间闪过一朵梦想的火花——如果有机会到开智去，哪怕只是短暂的实习，该有多好？照着头像插图画了偶像（阳老师）的画像，写下这一个不切实际的梦，贴在书桌前的墙上，天天仰望着。

望一两个星期之后，忽然萌生一个改变人生的念头——自己对线上教育行业这么感兴趣，怎么可以没有参加过偶像亲自指导的线上课程呢？于是，那一秒，就完成决定。

带着激动又兴奋的心情，我就这样来到开智这个学习“场”，信息分析三期。

一眨眼，七周课程就这样结束。在写这篇复盘时，心情很沉重，这份沉重包含了诸多遗憾与不舍；隐隐中，又带了一丝对未知的期待和兴奋，只因我相信，未来会因开智变得更好。

信息分析课程第三期，就这样结束了。对我而言，这却是一个新的开始。至少，我回顾发现，几个月前的自己似乎纠结于一个“非黑即白”的假象，逼着自己非在二选一的难题里作出抉择。参加课程之后，我为自己找到第三条路，这条路只因开智涌现——即便并非从事自己想要做的工作，领域相关就行，然后利用工作环境这一个“场”，学习内隐知识、持续创造自己在该领域的作品、成果，不断迭代，总有一天，这些堆叠起来的作品，便会化成通往梦想的道路。

开智，我们的缘分会一直持续下去……

复盘：没有人能替你走过任何一个难关

课程和作业回顾

  • Ch 1
    • 第一周理所当然地跟打了鸡血一样，充满动力和精力，兴奋异常。缺点是对于一个新进入的领域没有权重意识，对每个知识点、每个细节都投入了一样的注意力和精力，导致真正重要的地方没能有效花费时间去提升。实际上这个缺陷全程都持续，所以后面三周就不重复提了。
    • 理论部分理解得快，但实作部分天差地远。可见自己一直都特别不擅长把学到的东西过渡到实作部分。后面三周也有相同情况，不重复提。
  • Ch 2
    • 这是我在信分三大领域中最感兴趣的一个领域，因为自己对不少学科都非常感兴趣。跟康锋、榆贝、大蔚和舟舟队长等同学组成一个讨论群讨论。可惜自己贡献不大。后来，比起预期的完成两份进阶作业，只做完了进阶作业1，而且自己觉得不理想。不过，尽管还是有小小遗憾，初次尝试，已经可以算是合格。没学会如何使用CiteSpace。
    • 最有成就感的地方是看完了100篇文献的摘要。但实际上作用不大。希望日后可以持续练习，相比会越来越知道怎样从中抓到关键信息的感觉，也能够训练自己的权重意识。
  • Ch 3
    • 没有做进阶作业。基础作业也做得不理想。是整个课程里最弱的部分。也许复训时可以针对这点加强。或是等实际需要用到时，再好好学一遍。（可以刻意为自己创造一个需要用到的时机，写进阶作业就是其中一个这样的机会）
    • 这一个坑及生活中的茫然、压力让自己的激情消减，陷入课程时期的第一个也是最大的低谷。
  • Ch 4
    • 成功激励自己返回课程，当然也要感谢课程内容，因为相当于对一个极度口渴的人提供一口井，马上又燃烧起火焰~
    • 试图完成两个进阶作业，碍于必须尽快进入BP，没能达成。完成度还算满意，希望能够有机会真正完成。
    • 这周作业最大的收获是让自己更加了解自己，焦虑的内心平静下来。我把重点放在探索过去的自我了（为了更加了解自己），但更重要的部分、也是自己更感兴趣的部分，实际上是“可能的自我”，希望日后能够在这点持续迭代、探索，并写下来，为自己创造更多的机会。
    • 这周的结果就是，快速为自己找到了一份稳定的工作——虽然薪水不高，也是自己完全陌生、不曾感兴趣的学前教育工作不过，看了阳老师推荐的《无条件养育》成功拯救了我，让自己萌生“或许可以试一试也不错”的念头。尝试了也觉得并不如想象中坏。（好在离家不太远、有自己发挥和创造的空间、并且可以用以检验自己学到的学习理论）

主要收获 （未完待补 #TODO）

对我而言，阳老师和课程组用心打磨的信分课程内容很重要，课程以外的东西更重要。开智的课程有点神奇，因为除了课程内容的信息之外，课程外的信息也非常丰富，比如一些内隐知识，就值得我们用心留意，去观察、发现，然后吸收。

总结来说，结合课程内外的范围，我认识到以下自己认为关键、重要的概念：

信息分析的关键 认知的关键 输出的关键 其他技巧
全局认识 习惯大于目标 迭代意识，完成大于完美 用表格对资料降维
交叉验证 批量解决问题 以解决核心矛盾（问题）的思路输出 拆解问题
权重意识 内隐大于知识点 刻意练习 + 复盘 用卡片输出
4*5理论、框架意识 认知灵活性 最小行动 行动清单、检查清单降低认知负荷
时间-空间-变量关系 输出大于输入 最小产品

说实话，我并不是每个概念都真正了解，或有深刻认识。

因此，在这里就只挑其中几个来展开。

结语：明灯已指引我来到开智，故事便会改写，因为我愿意改变

也许我真的即愚昧个性又差，也没有良好习惯基础，不过明灯已经指引我来到开智了，那故事便会改写。因为，我愿意改变，也迫切期待着蜕变之后的自己（不管是蜗牛还是毛毛虫变成蝴蝶的隐喻）。我已经慢慢地从僵固型思维过渡到成长型思维。

我学会了“迭代”这一重要的概念。没有一个草图是一次就完美到位，一切都需要层层加叠，组合起来才是最终完整而美好的模样，而且，这个成品不是静止的，它将随着时间流动，经历一次次改进、升级，最终找到自己的最佳定位，得以展现最具优势、最出色的面貌——正如我自己的人生一样。

我是我自己的产品，而我又创造出许多产品。我最终会活成什么样的人生呢？我将用一辈子的时间探索着。

2018年年底能来到开智，跟开智小伙伴们一同跨年，真好！

我的不足之处

  1. 权重意识的缺陷明显，做作业、回答问题没抓到题眼，针对关键深入作答，以刻意练习所学知识、技能。比较多时候每个部分都希望做好、投入的心血也一样多，容易心力交瘁，却错过最关键部分的臻于至善。而那些最关键的部分，才是值得投注最多心血，让自己学到最多的地方。如同学在商业进阶作业对财报的学习等等。

  2. 做进阶作业时，老是不停掉坑，容易遇到行为瘫痪的时候。这些时候往往我都会花费大量（数小时到数天）时间跟坑死磕，也无法好好算计自己用的时间。不能进入心流，却还是逼迫自己毫无意义地消耗时间和精力。

  3. 虽然一直提醒自己完成大于完美了，还是有不少时候因为追求完美导致行为瘫痪。好在拥有优秀同侪的压力推动，这种情况已经改善很多，总算输出不少东西，尽管回头看可能都要摇头叹息，还是让人小小地高兴。

  4. 实作部分弱，需加强练习、多多练习、刻意练习。
  5. 作业常常写得比较看不到重点，自己写着、看着也晕。

日后改进

  1. 在开始一件事时，不管是工作、答题、读书或学习，要事先找到关键目标、环节，厘清不同重要程度的部分需要投入的资源，选定策略再开始做，以免陷入“看起来勤劳”的懒惰、虚假的勤劳，结果收获远远不如预期。
  2. 希望之后能够明确地用番茄时钟记录进出任务时间，如果卡壳两个番茄以上，就可以跳出来，先做其他事情，等到厘清思路之后再回去做，以免做毫无意义的浪费。也可以趁机参考别人如何做，为自己找到 参考、方向和灵感。
  3. 持续斟酌最小行动、最小产品。用卡片输出，在抓到问题的关键之后，先对整体进行一个全局的认识，建立框架，然后再一一针对单独的环节专注输出，适时回去调整框架。持续贯彻“完成大于完美”的原则，直到深刻地印在脑中，化为习惯。
  4. 完成行动清单、笔记之后，要多练习实作部分。（结合工作、自修计划、输出项目等等）
  5. 能用表格、图片表达的地方，就不要用落落长的文字表达，尽可能简化、视觉化。觉得内容太多了，也许试试看能不能限制在4*5的范围内？

行动清单

  • 在GitHub私人仓库里面，建一个信分课的工具箱（清单），把课程学到的信息分析工具统整、梳理一遍。
  • 继续完成课程一些比较重要的卡片笔记。
  • 把课程过程中未完成的行动清单给完成。
  • 从全局认识的视角出发，为自己未来两年的工作设计一个自修计划，以及设定输出项目。参考阳老师《给自己的教育》。
  • 了解如何书写周报、日报。
  • 了解如何使用行动清单。
  • 了解WOOP和SMART。
  • 写一篇人格心理学的综述。
  • 完成未完成的进阶作业，如商业分析、职业报告等。（或者复训再做？）
  • （课程结束之后）思考一下，我希望花时间了解全局的4/20个领域有哪些？
  • 为课程内容做几张mind maps。
  • 完成行动清单、笔记之后，要多练习实作部分。（结合工作、自修计划、输出项目等等）
